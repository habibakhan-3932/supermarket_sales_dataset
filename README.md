# Supermarket Sales Data Analysis

## Project Overview
This project performs Exploratory Data Analysis (EDA) on a supermarket sales dataset containing 1000 records and 17 columns.

## Tools Used
- Python
- Pandas
- Matplotlib
- VS Code

## Dataset Information
- Rows: 1000
- Columns: 17
- Missing Values: 0
- Duplicate Records: 0

## Key Findings

### 1. Transactions by City
- Yangon: 340
- Mandalay: 332
- Naypyitaw: 328

### 2. Product Line Popularity
- Fashion Accessories was the most popular product line.
- Health and Beauty was the least popular.

### 3. Total Sales by City
- Naypyitaw generated the highest sales revenue.

### 4. Payment Methods
- E-wallet was the most preferred payment method.
- Cash was a close second.

### 5. Gender Distribution
- Female: 501
- Male: 499

### 6. Customer Ratings
- Average Rating: 6.97/10
- Maximum Rating: 10
- Minimum Rating: 4

## Conclusion
The supermarket showed balanced customer activity across cities and genders. Naypyitaw generated the highest revenue, while Fashion Accessories was the most popular product category.
